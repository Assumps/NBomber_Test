## How To Use

To clone and run this application, you'll need [.Net 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) installed on your computer.

```
# Clone this repository
$ git clone https://github.com/Assumps/NBomber_Test.git

# Open the Solution in
$ \src\WheatherForecast\WheatherForecast.sln

# Run your docker desktop or docker engine

# Configure Multiple Startup Projects
$ is essencial to run NBomber and WeatherForecast API at the same time.

# Run the app

```
